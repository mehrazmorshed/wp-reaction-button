Get reaction buttons in single posts and pages of your WordPress website.

== Description ==

The WP Reaction Button plugin will display reaction buttons for logged-in users in single posts and pages of your WordPress website. You can also add reaction buttons via [reactions] shortcode and widget block for Gutenberg.

== Features ==
* Get Reaction Buttons after each single post
* Use Reaction Buttons  in single posts and pages
* Set Reaction Buttons using [reactions] shortcode
* Display Reaction Buttons only for logged-in users

== Installation ==

Install the WP Reaction Button plugin via WordPress Dashboard (from plugin directory or from your computer), or upload the files manually to your server and activate the plugin.

= Install via WordPress dashboard =

*From Plugin Directory*

* Visit ‘Plugins > Add New’
* Search for ‘WP Reaction Button’
* Install and Activate the plugin
* You will be redirected to the settings page

*From Your Computer*

* Download the WP Reaction Button plugin
* Visit ‘Plugins > Add New > Upload Plugin’
* Choose the plugin file in .zip format
* Click the Install Now button, and Activate after installation
* You will be redirected to the settings page

= Install via Uploading Files Manually =

* Download the WP Reaction Button plugin and unzip the folder
* Upload the folder to your ‘/wp-content/plugins/’ directory
* Activate the WP Reaction Button Plugin from Dashboard
* You will be redirected to the settings page

== Frequently Asked Questions ==

= How to enable reaction buttons? =

Activate the plugin. That's it.

= How to enable reaction buttons using shortcode? =

Use [reactions] shortcode to enable reaction buttons.

= How to enable reaction buttons using widget block of Gutenberg? =

Add a new block, search for WP Reaction Button and click on it to add.

= How to disable reaction buttons? =

Remove [reactions] shortcode, remove WP Reaction Button block and deactivate the plugin.
